Pour utiliser le script : 

python ScriptFinal.py 

IMPERATIF !!!!!

(ou modifier le script, la variable est tout en haut est ajouté avec l'option -cp de java dans le code)

Je vous invite donc à supprimer les fichiers dans le dossier data (sauf KTFGHU14.csv) 
et les fichiers dans models/model2 (sans supprimer les dossier !)
